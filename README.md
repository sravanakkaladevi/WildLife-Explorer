# WildLife Explorer

WildLife Explorer is a premium, beautifully designed frontend wildlife documentary blog and species directory. It was built as a clean, responsive, and highly interactive UI project.

## Features

- **Cinematic Hero Carousel:** Features a fully automated, infinitely looping image slider with smooth crossfade transitions to showcase featured wildlife.
- **Dynamic Typewriter Effect:** Provides engaging contextual lines describing each animal.
- **Wildlife Directory:** An interactive grid of 50 animal profiles dynamically generated from JavaScript data.
- **Search & Filter:** Allows users to easily search for specific animals or filter by their native regions/habitats.
- **Detailed Modals:** Users can view extended information about any species, including lifespan and conservation status.
- **Fully Responsive:** Carefully styled with vanilla CSS to provide a perfect viewing experience on all screen sizes.

## Technologies Used

- **HTML5:** Semantic and accessible page structure.
- **CSS3:** Custom styles, modern flexbox/grid layouts, smooth animations (`@keyframes`), and glassmorphism effects.
- **Vanilla JavaScript (ES6):** Handles dynamic data rendering, modal interactions, filtering, and the custom autoplay slider logic.

## Setup & Deployment

This project requires no backend or build tools. You can run it locally by simply opening the `index.html` file in your preferred web browser.

It is also fully ready to be deployed on **GitHub Pages** for live viewing.
